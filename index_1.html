Script para atualizar a planilha
Como usar: Copie o código abaixo, salve como atualizar.py na mesma pasta que Financeiro_Terapeuta.xlsx e execute com python atualizar.py
Não sabe usar o terminal? Veja o passo a passo
  1. Certifique-se de ter Python instalado (python.org)
  2. Instale a biblioteca: pip install openpyxl
  3. Salve atualizar.py na mesma pasta de Financeiro_Terapeuta.xlsx
  4. Abra o terminal nessa pasta: python atualizar.py
